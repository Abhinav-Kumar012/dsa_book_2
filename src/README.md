# The Definitive DSA Interview Book

> A complete Data Structures & Algorithms textbook designed specifically for coding interview preparation.

## Who This Book Is For

This book is written for software engineers preparing for coding interviews at top technology companies. It assumes no prior knowledge of advanced mathematics and builds every concept from first principles.

## How to Use This Book

- **New to DSA?** Start from Chapter 1 and proceed sequentially.
- **Brushing up?** Jump to specific chapters using the table of contents.
- **Interview in 30 days?** Follow the [30-Day Crash Course](appendices/30-day-crash-course.md).
- **Interview in 60 days?** Follow the [60-Day Study Plan](appendices/60-day-study-plan.md).
- **Interview in 90 days?** Follow the [90-Day Study Plan](appendices/90-day-study-plan.md).

## Book Structure

| Part | Chapters | Focus |
|------|----------|-------|
| I: Foundations | 1–7 | Math, Complexity, Arrays, Strings, Sorting, Searching, Binary Search |
| II: Core Data Structures | 8–14 | Hashing, Recursion, Backtracking, Stacks, Queues, Linked Lists, Trees |
| III: Advanced Data Structures | 15–21 | BST, Heaps, Trie, DSU, Segment Tree, Fenwick Tree, Sparse Table |
| IV: Graphs | 22–28 | DFS, BFS, Topological Sort, Shortest Paths, MST, Advanced Graphs, Network Flow |
| V: Algorithm Paradigms | 29–33 | DP Fundamentals, DP Patterns, Greedy, Bit Manipulation, Two Pointers |
| VI: Problem-Solving Patterns | 34–39 | Sliding Window, Prefix Sum, Difference Array, Monotonic Stack, Monotonic Queue, Binary Lifting & LCA |
| VII: String Algorithms | 40–45 | Rolling Hash, KMP, Z Algorithm, Trie Applications, Suffix Array, Suffix Automaton |
| VIII: Interview Preparation | 46–50 | Problem Solving, Technical Communication, Behavioral, Mock Interviews, Final Revision |
| Appendices | A1–A12 | STL Guide, Cheat Sheets, Templates, Study Plans, Company-wise Prep |

- Meaningful variable names
